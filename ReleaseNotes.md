# ReleaseNotes

This document summarizes the publicly released versions of the OpenHealth Requirements Gradle plugin.

## Release Overview

| Version | Release Date | Highlights                                                                                                          |
|--------|--------------|---------------------------------------------------------------------------------------------------------------------|
| 1.0.0  | 2025-04-16   | Initial public release of the plugin |
| 1.0.1  | 2025-03-05   | Gradle Plugin publishing                                                                                    |

## Change Log

### 1.0.0
- Extracts structured requirements from code comments
- Supports configurable comment prefixes (e.g., //, #, etc.)
- Works with any file type
- Outputs results as a CSV file

### 1.0.1
- Gradle Plugin publishing.
